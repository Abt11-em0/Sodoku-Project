# Sodoku-Puzzle-Generator 
Aliver Griffin-Mencia 
This is a Sodoku finished puzzle board generator which creates a finished puzzle for the game sodoku that can be used to create an unfinished puzzle.
You can run this using any java console.
The board is generated via cycling through a 4D array to fill in valid numbers and changing 2D arrays within the 4D array when no valid options are left.
